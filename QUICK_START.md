# Quick Start Guide

## Start in 3 Steps

### 1. Start Backend
```bash
cd AutoBackend
npm start
```

### 2. Start Frontend
```bash
cd shubh-hyundai-dashboard
npm run dev
```

### 3. Access Dashboard
- URL: http://localhost:3000/login
- Login: sm.pune@shubh.com / password

## Upload Excel Files

1. Select data type from dropdown
2. Choose Excel file
3. Click Upload
4. View data in tabs

## Excel Format

### RO Billing
Bill Date | Service Advisor | Labour Amt | Part Amt | Work Type

### Booking List
Service Advisor | B.T Date & Time | Work Type | Status

### Warranty Claims
Claim Date | Claim Type | Status | Labour | Part

### Operation Wise
OP/Part Desc. | Count | Amt

## Features

- Automatic averages calculation
- Work Type Breakdown with pie chart
- All data tables with proper columns
- Refresh and Reset functionality
- Modern, responsive UI
